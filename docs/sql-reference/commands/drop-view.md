# DROP VIEW

Deletes a view.

## Syntax

```sql
DROP VIEW [IF EXISTS] <view_name>
```

| Parameter     | Description                         |
| :------------- | :----------------------------------- |
| `<view_name>` | The name of the view to be deleted. |
